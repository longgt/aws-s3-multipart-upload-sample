# aws-s3-multipart-upload-sample
Sample how to upload to AWS S3 using multipart upload for AWS S3 Javascript SDK

# How to begin

Copy .env.example file into .env

Change the value in there to match with your configuration

Run the below command to start

```
npm run start
```

It will upload `data\sample.png` into your AWS S3 bucket at `multiparts` folder.
